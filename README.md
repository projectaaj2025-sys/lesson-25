# 📊 Урок 25: Обсуждение результатов и оформление работы

Интерактивный тест с 3 вариантами на тему обсуждения результатов исследования, оформления научной работы, устных докладов и презентаций.

## 📁 Файлы проекта

```
urok25-test/
├── index.html              # Страница учеников (CSS + JS встроены)
├── teacher.html            # Панель учителя (CSS + JS встроены)
├── google-apps-script.js   # Код для Google Apps Script (тот же, что в Уроке 17)
└── README.md
```

## 🚀 Быстрый старт

### 1. Создайте Google Таблицу
- [sheets.new](https://sheets.new) → скопируйте ID из URL

### 2. Создайте Google Apps Script
- [script.google.com](https://script.google.com) → Новый проект
- Вставьте код из `google-apps-script.js` (из Урока 17 — он универсальный)
- **Развернуть** → **Веб-приложение** → Доступ: **Все**
- Скопируйте URL веб-приложения и замените `APPS_SCRIPT_URL` в `index.html` и `teacher.html`

### 3. Загрузите на GitHub Pages
- Создайте репозиторий, загрузите `index.html` и `teacher.html`
- Settings → Pages → main branch, / (root)

## 🎮 Типы заданий (с автопроверкой)

| Тип задания | Описание | Баллы |
|-------------|----------|-------|
| **Соответствие (matching)** | Drag-and-drop: соединить элемент с характеристикой | 3 |
| **Корзины (buckets)** | Drag-and-drop: распределить по группам | 4 |
| **Заполнение пропусков (cloze)** | Выпадающие списки с правильными вариантами | 2–4 |
| **Множественный выбор (checkbox)** | Отметить несколько верных вариантов | 3 |
| **Одиночный выбор (radio)** | Выбрать один верный вариант | 3 |
| **Упорядочивание (ordering)** | Drag-and-drop: расставить в правильном порядке | 4 |
